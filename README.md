# titanic-kaggle
Titanic survival prediction. Kaggle score: 0.779
In this project I've predicted the number and percentage of survivors of the wreck of the Titanic. 
The dataset "Titanic - Machine Learning from Disaster" is the most popular dataset on Kaggle for the first project for beginning to the Data Science and Data Analysis.
In this project firstly I conducted exploratory data analysis, then did feature engineering to increase the accuracy of prediction, then dropped unnecessary data. 
I've used and trained the Random Forest Classifier. 
Then I predicted the number and percentage of survivors from test dataset. 
I have got 82% of accuracy and get 77,99 score on Kaggle.
